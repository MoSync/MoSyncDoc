MoSync Panics Reference (maPanic codes)

A "panic" is an unrecoverable error. The MoSync API will throw a panic in a range of situations. Your application code should always handle the situations that might throw a panic, for example insufficient memory, indexes out-of-bounds, division by zero, and read/write failed.
